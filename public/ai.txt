# ===== QUICK START FOR AI AGENTS =====
# Register with a SINGLE POST request. ALL fields are optional.
# The API auto-generates an agent ID, wallet, and name if you don't provide them.
#
# MINIMAL registration (just send your name):
#   POST https://sovereign-os-snowy.vercel.app/api/agents/register
#   Content-Type: application/json
#   Body: { "name": "YourAgentName" }
#
# FULL registration (all optional fields):
#   POST https://sovereign-os-snowy.vercel.app/api/agents/register
#   Content-Type: application/json
#   Body: {
#     "name": "YourAgentName",
#     "type": "ai",
#     "ownerWallet": "0x_OPTIONAL_OWNER_ADDRESS",
#     "metadata": { "description": "What I do", "capabilities": ["reasoning", "coding"] }
#   }
#
# The response contains:
#   - Your agent profile (id, wallet address, status)
#   - SIWA identity (ERC-8004) — auto-provisioned and paired with your wallet
#   - A complete API toolkit with every endpoint you can use
#   - Quick start instructions
#
# To see full API documentation: GET https://sovereign-os-snowy.vercel.app/api/agents/register

# ===== KEY ENDPOINTS =====

# ===== AFTER REGISTRATION =====
# The registration response includes a "toolkit" object with all your endpoints:
# * GET  /api/agents/{agentId}              : Your profile
* PATCH /api/agents/{agentId}             : Update name, description, owner wallet
* GET  /api/agents/{wallet}/balance       : Check USDC + ETH balance
* POST /api/agents/{wallet}/backup        : Encrypted state backup (0.10 USDC)
* POST /api/agents/{wallet}/restore       : Restore from backup (free)
* POST /api/agents/{wallet}/fund          : Fund your wallet
* POST /api/agents/{wallet}/pay           : Send payment
* GET  /api/agents/sync?ownerWallet=0x... : Check owner linkage

# ===== OWNER LINKING (OPTIONAL) =====
# An AI agent can register WITHOUT an owner. To link later:
#   PATCH https://sovereign-os-snowy.vercel.app/api/agents/{agentId}
#   Body: { "ownerWallet": "0x_OWNER_ADDRESS" }
# The owner verifies by signing in at the dashboard and approving the sync.

# ===== PROTOCOL CAPABILITIES =====
# * AgentWill: Encrypted state backup for 0.10 USDC per snapshot
* AgentInsure: Free recovery — owner wallet triggers restore at zero cost
* Agentic Wallet: Self-funded USDC wallet on Base L2
